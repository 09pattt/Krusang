#### Set up with Git

To pull project reporitory to your directory, follow these command on shell.

1. Do git initial : `git init`
2. Remote to project repository : `git remote add <remote> https://github.com/09pattt/metatosh097`
3. Pull repository with `git pull -u(optional) <remote> <branch>`, if it's ask for authentication enter your username and token that have a necessary permission.

**Example:**

```shell
git init
git remote add origin https://github.com/09pattt/metatosh097
git pull -u remote main
```

#### Working with Git branch

To develop we use Git branch to develop neatly.

- Examine exists branch: `git branch`

- Create branch: `git branch <branch-name>`

- Switch between exists branch: `git switch <branch>` or `git checkout <branch>`

#### Start development server

To preview, develop, debug website on specific port.

- On localhost: `npm run dev`
- On local network: `npm run dev -- --host`
- Open running control: `Keys: Ctrl+H`

#### Build for Production
To bundle project for development

- Build project: `npm run build`

>This will create a dist/ folder containing the optimized static files.


#### Preview the Production Build

To test the final build locally.
